# Profile Authentication Project

A **MERN stack** application with **JWT-based authentication and authorization** for secure user profile management.

---

## Features

- User registration and login with email and password.
- Secure password hashing.
- JWT authentication & authorization for secure sessions.
- User profile management (view and update profile).
- Responsive UI built with **React** and **Tailwind CSS**.

---

## Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT (JSON Web Tokens)  
- **Version Control:** Git & GitHub
